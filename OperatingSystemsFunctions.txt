Starting up a computer or mobile device
Power gets thruogh the circuitry via electrical current
Tests are then executed based on the system set-up including peripharals, making sure everything is working properly
If everything checks out, the kernal of the OS loads to the RAM
OS in the memory then controls the device and loads configurations, UI then shows up on screen and so forth 

User Interface (UI)
This is what you see and how you use a compuer or device
There are two types of UI
Graphical user interface (GUI) and command-line interface

Configuring devices
Peripharals connected to a computer or device initiates a driver
Driver is a progam that communicates to the OS when a specific device is connected
A driver is and must be installed for each connected device for it to work properly
